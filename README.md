# Inventory Management System (MERN)

## Overview
This is a basic full-stack Inventory Management System built using the MERN stack.

## Features
- Admin & Employee Login
- Add / Delete Products
- View Stock
- JWT Authentication

## Tech Stack
- React (Frontend)
- Node.js & Express (Backend)
- MongoDB (Database)
